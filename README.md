# Doro

> A pomodoro web-application integrated with a Trello board for project and time management

## Tech Stack

- PostgreSQL (node-postgres)
- Express.js
- React.js (TypeScript)
- Node.js
- NextUI components
- TailwindCSS
- Zustand state management
- OAuth 2.0
- Jest unit testing
